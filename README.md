# Count in Array

## Даалгавар

`index.js` дотор байгаа `count` функцыг дуусга.

count функц нь параметраар string авах бөгөөд энэ string нь таслалаар тусгаарлагдсан натурал тоон дараалал байна тийм бол хамгийн их давтагдсан тоог буцаа. хэрвээ ийм тоо олон бол хамгийн бага утгатайг буцаана


Жишээ нь:
* `"1,2,3,7,7,8,1,100"` - ийг => `1` гэж буцаана
* `"2,7,2,9,7,1,100,7"` - ийг => `7` гэж буцаана
